课前准备与热身

【调查】
上网调查一下现在主流的编程语言有哪些，简要了解一下挑选一个自己喜欢的编程语言
(最好有别于课程内将要讲解的Java与Javascript)

【动手】
下载并安装
  - Chrome浏览器
  - Visual Studio Code
  - Git
尝试用自己挑选的编程语言编写并运行"Hello World"程序
创建自己的Github账号，并创建自己的第一个Github Project
将自己写好的"Hello World"程序提交到创建的Project中，将链接分享给大家

【要求】
动手部分没做成功也不必担忧，尽力而为。
有能力的话也可以不限于上述要求，鼓励自行发挥。
